# AGENTS.md — Landing CEO Aumentado

## Objetivo del proyecto
Crear una landing page profesional para vender el libro "CEO Aumentado" de Brian Pajares, capturar leads con Supabase y dirigir tráfico hacia Amazon KDP.

## Stack actual
- HTML
- CSS
- JavaScript vanilla
- Assets locales en /assets
- Supabase para leads y eventos
- Deploy final en Vercel o Netlify

## Reglas críticas
1. Mantener el diseño premium, ejecutivo, tecnológico y elegante.
2. No eliminar el CTA principal hacia Amazon.
3. No publicar el archivo Word del libro como descarga pública.
4. No usar service_role key de Supabase en frontend.
5. Usar solo Supabase publishable key o anon public key.
6. Mantener responsive design para mobile, tablet y desktop.
7. Agregar captura de leads con consentimiento.
8. Registrar eventos: page_view, lead_submit y amazon_click.
9. Mantener el sitio rápido, ligero y compatible con hosting estático.
10. No romper index.html ni styles.css.

## Objetivo comercial
Convertir visitantes en:
- lectores interesados,
- leads para futuras campañas,
- compradores potenciales en Amazon.

## Elementos obligatorios
- Hero section fuerte.
- Imagen de portada.
- Beneficios del libro.
- Sección de lead magnet.
- Formulario con nombre, email, país y consentimiento.
- Botón a Amazon.
- Contraportada o mockup visual.
- Sección sobre el autor.
- Footer con aviso básico de privacidad.
